TOGGLE STATES:
─────────────────────────────────────────────────────────────────────────────
STATE 1: Both Visible (Default)
  - Original: alpha=0.4, linewidth=1.5 (light, transparent)
  - Processed: alpha=0.9, linewidth=2.0 (solid, prominent)
  - Both use same color (curve's industry color)

STATE 2: Original Only
  - Original: alpha=0.9, linewidth=2.0 (fully visible)
  - Processed: hidden

STATE 3: Processed Only  
  - Processed: alpha=0.9, linewidth=2.0 (fully visible)
  - Original: hidden

UX BENEFITS:
- Direct visual comparison on same scale
- Easy to see exactly where changes occurred
- Toggle allows focused view of either dataset
- Statistics panel shows numerical comparison
- Opacity distinction (not color) maintains visual coherence
